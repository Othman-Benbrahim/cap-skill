# cap
CAP est un skill d'orientation et de résilience durable. Il prend la suite d'un travail d'analyse (Miroir Intégral) ou de transformation (SEUIL) pour aider la personne à construire une direction de vie alignée sur ses valeurs profondes, ancrer les changements dans le quotidien, et bâtir une résilience qui tient dans le temps.
